# System Sentinel Audit

A lightweight Bash utility designed to perform a quick health check on a Linux system. This project marks the completion of the "Foundations" phase of the Linux Bible (Chapters 1-5).

## 🚀 Features
- **Kernel Verification:** Pulls the current system release info.
- **Storage Monitoring:** Lists the top 5 partition usages.
- **Memory Check:** Reports free and used RAM in megabytes.
- **Automated Reporting:** Outputs all data to a timestamped `report.txt`.
